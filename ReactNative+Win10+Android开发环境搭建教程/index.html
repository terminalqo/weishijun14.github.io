2018 React Native + Win 10 + Android 开发环境搭建教程

楼主搭个android环境用了7-8个小时，你敢信 ???

废话不多说，进入主题：

1. expo XDE :

expo XDE是 React Native 的桌面端环境。 这个没什么太多好说的。 根据官网指导， 下载下来之后，安装即可。 官网地址： https://docs.expo.io/versions/latest/introduction/installation

2. Genymotion :

genymotion 是pc端的 android 模拟器。这里是free版的下载地址： （在下载之前您得首先在官网注册一个账号。） https://www.genymotion.com/fun-zone/ 同样也是下载完成之后安装。

3. JDK :

可以进入http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html 选择对应的版本下载。这里我下载 Windows x64 这个版本。 下载完成之后，安装，验证是否安装成功。配置JAVA环境变量。（这些网上都有，我就不在此重复了。。）

4. android studio :

android studio 下载地址： https://developer.android.com/studio/index.html 这是另外一个下载地址： http://www.androiddevtools.cn/

建议用浏览器直接从官方原始链接下载，不要用迅雷下载、不要用迅雷下载、不要用迅雷下载，重要的事情说三遍，避免类似XCodeGhost的事件！！！

下载完 android studio 之后开始安装（这里是需要注意的地方）：

点击finish，之后：

这里会出现报错，让你选择一个代理，无视，点cancel.

要点1： 勾选Android virtual device (742MB), 要点2： 选择你的Android SDK 位置。这里我没有使用默认的位置， 自己选了一个单独的位置。然后点击下一步。

*装完之后点击configure, 选择sdk manager。 *

进入之后，第一步，设置代理，这是为了下载sdk的时候更加快。 （代理地址http://www.androiddevtools.cn/ 【你也可以自己找一个】）

第二步，设置sdk。sdk地址设置为刚刚在安装的时候选的位置。然后在下面的区域中选择你需要安装的sdk，我自己选的是从最新的8.1 一直到6.0

*第三步，点击sdk tools 标签，在下面的区域中选择android sdk build – 23.0.1, 勾选 show package details, 选择Android Support Repository。 最后点击ok ，进行sdk 安装。 *

5. Android SDK :

安装好 SDK 和相应的工具之后，建议进行 SDK 相关的环境变量的配置。

搞定以上步骤之后，基本上是完成了第一阶段的配置工作，下面来到第二阶段 ：）

1. 打开 genymotion, 在settings中的 account 中 登陆你之前在官网注册的账号。

2. 切换到ADB中， 设置你的Android SDK 地址为 你之前在 android studio 中的 sdk 路径。

3. 完成设置，然后点击Add. 加入你想要的设备以及android版本。完了之后，点击start, 启动设备。

启动之后是这样的：

4. 打开Expo XDE， 选择一个project, 运行。

看到如图，左下角的红框内的字样 ， 说明启动成功，然后点击右上角的 open on android,在模拟器内打开。

注意，有99%的可能，你会看见报错：

Couldn’t start project on Android: could not install smartsocket listener: cannot bind to 127.0.0.1:5037: Only one usage of each socket address (protocol/network address/port) is normally permitted. (10048) could not read ok from ADB Server * failed to start daemon * error: cannot connect to daemon

解决办法：

  1. Close XDE, Expo CLI, and any other open command line terminals. They only pick up changes to Environment Variables upon opening. 关闭你的XDE, genymotion, 打开环境变量的配置窗口。
  2. 选择path ， 打开编辑窗口。
  3. 新建一个变量。
  4. 路径就是你genymotion的安装目录下的tools目录。
  5. 然后确定确定确定….
  6. 重新打开XDE, Gentmotion, 打开模拟器， 打开project， 然后再 open on android。

到此为止，结束。
